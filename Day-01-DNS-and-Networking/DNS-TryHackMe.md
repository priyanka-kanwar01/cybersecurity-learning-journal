## 🧠 What I Learned Today

### 🔹 What is DNS?
DNS (Domain Name System) translates human-readable domain names into IP addresses so browsers can locate servers.

### 🔹 Why DNS is Important?
- Humans remember names, not IP addresses
- Enables access to websites and online services
- Core component of the internet

### 🔹 DNS Record Types
- **A** – Domain to IPv4
- **AAAA** – Domain to IPv6
- **CNAME** – Alias record
- **MX** – Mail servers
- **TXT** – Verification & security

### 🔹 DNS Resolution Process
1. User enters domain
2. Query goes to resolver
3. Root → TLD → Authoritative server
4. IP returned to browser

## 🛡️ Cybersecurity Relevance
- Used in phishing attacks
- DNS spoofing & poisoning risks
- Important for threat detection
