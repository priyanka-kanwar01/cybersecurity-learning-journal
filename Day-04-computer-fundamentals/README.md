Day 4 — Cybersecurity Journey 💻🔐

Today I focused on Computer Fundamentals, which are essential for building a strong foundation in cybersecurity and penetration testing.

Topics I Learned
CPU, RAM & Storage — Understanding how computers process, store, and manage data.
BIOS/UEFI — Learning how systems boot and initialize hardware.
Processes & Threads — How operating systems handle running applications and multitasking.
File Systems — Understanding how data is organized and stored.
Virtualization — Basics of virtual machines and why they are important in cybersecurity labs.
Networking Basics — IP addresses, ports, TCP/UDP, and communication between devices.
Localhost & Loopback — How systems communicate internally.
Kernel Basics — The core part of an operating system and its role in security.
Key Takeaways
Cybersecurity is not just about hacking tools — understanding how computers actually work is critical.
Networking and operating system knowledge form the backbone of penetration testing.
Virtualization is extremely useful for creating safe testing environments.
Knowing system internals helps in understanding vulnerabilities and attack surfaces.
Progress

Every day I’m improving my understanding of the technology behind cybersecurity. Building strong fundamentals now will make advanced topics like web exploitation, Active Directory attacks, privilege escalation, and red teaming much easier later.
