## 🧠 What I Learned

### 🔹 What is HTTP?
HTTP (HyperText Transfer Protocol) is the protocol used for communication between a web browser (client) and a web server.

It allows users to request and receive web pages and other resources from servers.

### 🔹 HTTP vs HTTPS
- **HTTP** – Data is transmitted in plain text
- **HTTPS** – Encrypted communication using SSL/TLS

HTTPS is more secure and protects data from attackers.

### 🔹 HTTP Request
When a client wants a resource, it sends an HTTP request containing:

- Request Method
- URL
- Headers
- Body (optional)

Example request methods:
- **GET** – Retrieve data
- **POST** – Send data to server
- **PUT** – Update data
- **DELETE** – Remove data

### 🔹 HTTP Response
The server sends back a response containing:

- Status Code
- Headers
- Response Body

Example status codes:
- **200** – OK
- **301** – Redirect
- **404** – Not Found
- **500** – Server Error

### 🔹 Why HTTP is Important in Cybersecurity
Understanding HTTP is essential because many attacks happen through web applications.

Examples:
- SQL Injection
- Cross-Site Scripting (XSS)
- Session Hijacking

Security professionals analyze HTTP traffic to detect vulnerabilities.
